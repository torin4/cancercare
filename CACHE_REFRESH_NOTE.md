# Browser Cache Issue

The build has been fixed and is correct. The error you're seeing is due to browser cache.

## Solution

**Please do a hard refresh to clear the browser cache:**

- **Mac**: `Cmd + Shift + R` or `Cmd + Option + R`
- **Windows/Linux**: `Ctrl + Shift + R` or `Ctrl + F5`

Or manually clear your browser cache and reload the page.
