# Cursor project config (CancerCare)

## Context7 MCP (React docs)

This project uses [Context7](https://context7.com) for up-to-date React and library docs. It's configured **only for this project**; other workspaces use your global Cursor MCP config.

### Setup

1. Copy `mcp.json.example` → `mcp.json`.
2. Get an API key from [context7.com/dashboard](https://context7.com/dashboard).
3. In `mcp.json`, replace `REPLACE_WITH_YOUR_CONTEXT7_API_KEY` with your key.
4. Reload Cursor (e.g. **Developer: Reload Window**).

`mcp.json` is gitignored so your API key is never committed.
